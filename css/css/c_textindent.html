text-indent版本：CSS1　 兼容性：IE4+ NS4+　继承性：无
语法：
text-indent : length
取值：
length :百分比数字 | 由浮点数字和单位标识符组成的长度值，允许为负值。请参阅 长度单位
说明：
检索或设置对象中的文本的缩进。默认值为 0 。
在被另一个对象(如 br )断开的对象内不能应用本属性。
此属性对于 currentStyle 对象而言是只读的。对于其他对象而言是可读写的。
对应的脚本特性为 textIndent 。
示例：
DIV { text-indent:2cm }
.click1 { text-indent:50% }
.click2 { text-indent: }
