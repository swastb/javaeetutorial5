Emboss兼容性：IE5.5+
语法：
filter : progid:DXImageTransform.Microsoft.Emboss ( enabled=bEnabled , Bias=fBias )
属性：
enabled : 可选项。布尔值(Boolean)。设置或检索滤镜是否激活。true | false
true :默认值。滤镜激活。
false :滤镜被禁止。
Bias : 可选项。浮点数(Float)。设置或检索添加到滤镜结果的每种颜色组分的值的百分比。取值范围为 -1.0 - 1.0 。默认值为 0.7 。大的此参数值产生高亮滤光效果。高对比度的图片受此滤镜的影响较小。
特性：
Enabled :可读写。布尔值(Boolean)。参阅 enabled 属性。
Bias :可读写。浮点数(Float)。参阅 Bias 属性。
说明：
用灰度值为对象内容制作浮雕纹理效果。
示例：
#idDiv{position:absolute; left:140px; height:400; width:400;filter:progid:DXImageTransform.Microsoft.emboss(enabled='false') ; }
